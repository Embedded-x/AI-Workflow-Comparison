# Project Rules

## Tech Stack

- HTML
- CSS
- JavaScript
- Cursor IDE

## Coding Rules

1. Validate all user inputs.
2. Use semantic HTML elements.
3. Write readable and modular JavaScript.
4. Keep CSS organized.
5. Verify AI-generated code before accepting it.
6. Follow Conventional Commits.

## Additional Project Rules

1. Always ask the AI to create an implementation plan before generating code.
2. Review AI-generated code manually instead of accepting it immediately.
3. Validate all user inputs and test common edge cases before committing changes.
4. Prioritize semantic HTML and accessibility.
5. Verify that every requirement from the prompt has been implemented.
